[noip模拟赛] 礼物gift

Description

夏川的生日就要到了。作为夏川形式上的男朋友,季堂打算给夏川买一些生日礼物。

商店里一共有种礼物。夏川每得到一种礼物,就会获得相应喜悦值Wi(每种礼物的喜悦值不能重复获得)。
每次,店员会按照一定的概率Pi(或者不拿出礼物),将第i种礼物拿出来。
季堂每次都会将店员拿出来的礼物买下来。没有拿出来视为什么都没有买到,也算一次购买。
众所周知,白毛切开都是黑的。所以季堂希望最后夏川的喜悦值尽可能地高。
求夏川最后最大的喜悦值是多少,并求出使夏川得到这个喜悦值,季堂的期望购买次数。

Input

第一行,一个整数N,表示有N种礼物。
接下来N行,每行一个实数Pi和正整数Wi,表示第i种礼物被拿出来的概率和可以获得喜悦值。

Output
第一行,一个整数表示可以获得的最大喜悦值。
第二行,一个实数表示获得这个喜悦值的期望购买次数,保留3位小数。
Sample Input
3
0.1 2
0.2 5
0.3 7
Sample Output
14
12.167

HINT

对于10%的数据,N=1N=1N=1
对于30%的数据,N≤5N\leq 5N≤5
对于100%的数据, N≤20N\leq 20N≤20 , 0<Wi≤1090 < W_{i} \leq 10^{9}0<Wi​≤109 , 0<Pi≤10 < Pi ≤ 10<Pi≤1 且 ∑Pi≤1\sum P_{i} \leq 1∑Pi​≤1

关键在于推公式

在做思考这个题之前先想一个比较简单的问题：

平均需要扔多少次硬币才能够得到连续2个正面？

要先知道一般的期望计算的方法：元素的期望=元素的概率*元素的值
这个问题应该把平均的次数看成元素的值
可以设平均需要\(T\) 次才能得到连续2个正面
写出方程：
T=(1−p)(1+T)+p(1−p)(2+T)+2p2T=\left(1-p \right)\left(1+T \right)+p\left(1-p \right)\left(2+T \right)+2p^{2}T=(1−p)(1+T)+p(1−p)(2+T)+2p2 (ppp为投到正面的概率)
中间的(1−p)(1+T)\left(1-p \right)\left(1+T \right)(1−p)(1+T)代表如果第一次投到反面，那么还平均需要TTT次才能得到连续2个正面所以需要1+T1+T1+T次
p(1−p)(2+T)p\left(1-p \right)\left(2+T \right)p(1−p)(2+T)代表如果第一次投到正面第二次投到反面那么还平均需要TTT次才能得到连续2个正面所以需要2+T2+T2+T次
2p22p^{2}2p2代表如果两次都投中的期望
化解即可求得这个问题的解

再来看这个题目
用kkk是一个二进制数表示买到了哪些物品 例如:k=101101k=101101k=101101 表示买到了第1,3,4,6个物品
fff表示期望值
假设已经求到了所有比kkk少一个物品的期望，k′k'k′表示比kkk少一个物品的二进制数，ppp为每个物品拿出来的概率，iii表示从k′k'k′这个状态还需要买第iii个物品就可以变成kkk，那么公式可以写成:

fk=∑pi⋅fk′+(1−∑pi)fk+1f_{k}=\sum p_{i}\cdot f_{k'}+(1-\sum p_{i})f_{k}+1fk​=∑pi​⋅fk′​+(1−∑pi​)fk​+1

(1−∑pi)fk(1-\sum p_{i})f_{k}(1−∑pi​)fk​表示之前kkk里面包含的物品一个都没有买到，那么需要再买fkf_{k}fk​次才能卖完kkk个
